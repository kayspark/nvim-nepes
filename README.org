* Installation

1. Add to lazy.nvim:
#+begin_src lua
{ 'kayspark/nvim-nepes', opts = { theme = 'dark' } }
#+end_src

* Configuration

#+begin_src shell
vim.cmd("colorscheme nepes")
#+end_src
